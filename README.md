# SaveInfo for VS code
## Author
Кулагин Никита Олегович М3102

## Description

This plugin shows the change in number of characters and lines after save in a text document.

## Commands

- `SaveInfo: turn on`

    Turns on the plugin.

- `SaveInfo: turn off`

    Turns off the plugin.

- `SaveInfo: toggle`

    Toggles the plugin on/off.

You can access these commands from the command palette.
